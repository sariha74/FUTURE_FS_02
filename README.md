🚀 Mini CRM (Client Lead Management System)

A simple yet powerful **Full Stack Mini CRM** built using **Node.js, Express, MongoDB, and Vanilla JavaScript**.  
This project helps manage leads, track their status, and maintain notes efficiently.



✨ Features

- 🔐 Simple Login System (Token-based authentication)
- ➕ Add new leads (Name, Email, Source)
- 📊 Track lead status (New / Contacted / Converted)
- 📝 Add and update notes for each lead
- 📋 Dynamic dashboard with real-time updates
- 📈 Live stats (Total, Contacted, Converted)
- 🗑️ Delete leads
- 🎨 Modern UI with glassmorphism & hover effects



🛠️ Tech Stack

 Frontend:
- HTML5
- CSS3 (Glassmorphism UI + Animations)
- JavaScript (Fetch API)

 Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)
